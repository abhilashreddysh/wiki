# Homelab Infrastructure

This documentation covers every part of my homelab, from physical hardware to
Kubernetes workloads.

The objective isn't simply to self-host services.

It's to build infrastructure that is reproducible, observable, automated, and
easy to recover when something inevitably fails. Because eventually it will.
